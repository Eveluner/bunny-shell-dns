## ✨ 核心功能

### ✅ DNS 区域管理
```
✓ 列表所有区域（表格式展示）
✓ 创建新区域（带域名验证）
✓ 删除区域（二次确认保护）
✓ 区域详情查看
```

### ✅ DNS 记录管理
```
✓ 列表区域内的所有记录
✓ 添加新记录（8 种类型支持）
✓ 更新现有记录
✓ 删除记录（操作确认）
```

### ✅ 支持的 DNS 类型
```
A (1)          - IPv4 地址
AAAA (28)      - IPv6 地址
CNAME (5)      - 规范名称
MX (15)        - 邮件交换
TXT (16)       - 文本记录
NS (2)         - 名字服务器
SRV (33)       - 服务记录
CAA (257)      - 证书授权
```

### ✅ 验证和安全
```
✓ IPv4 地址格式验证
✓ IPv6 地址格式验证  
✓ 域名格式验证
✓ 数字输入验证（TTL、优先级）
✓ 非空值检查
✓ 二次删除确认
✓ API Key 环境变量管理
```

## 🎯 使用场景

### 场景 1: 日常 DNS 管理
```bash
# 用户场景：域名管理员
./bunnydns.sh
# 1. 查看区域 & 11. 管理记录
→ 添加 / 更新 / 删除记录
→ 验证更改
```

### 场景 2: 自动化部署
```bash
# 用户场景：DevOps 工程师
BUNNY_API_KEY="key" ./bunnydns.sh <<EOF
5
zone_id
CNAME
app
app.example.com
3600
EOF
```

### 场景 3: 批量操作
```bash
# 用户场景：批量配置
for domain in domains.txt; do
    ./bunnydns.sh # 逐个添加
done
```

### 场景 4: 备份和恢复
```bash
# 用户场景：灾难恢复
# 使用提供的脚本定期备份 DNS 配置
./bunnydns.sh > dns_backup.json
```

- Bash 4.0+
- curl
- Linux / macOS / Windows (WSL)

```bash

brew install curl jq

sudo yum install -y curl jq
```
